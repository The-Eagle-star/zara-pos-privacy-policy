Welcome to ZaraPOS.

ZaraPOS is a Point of Sale (POS) and business management application developed by Zafrionet Ltd. This Privacy Policy explains how we collect, use, store, and protect information when you use our application and related services.

We use collected information to:

4 Firebase Services

ZaraPOS uses services provided by Google Firebase, including:

Firebase Authentication

Used for secure user login and account management.

Cloud Firestore / Firebase Database

Used to securely store business, inventory, customer, supplier, and transaction data.

Firebase Analytics

Used to understand app usage and improve user experience.

Firebase Crashlytics

Used to monitor crashes and application performance issues.
